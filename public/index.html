Servicio de Recuperación de Contraseña

Para restablecer tu contraseña, usa el enlace del correo electrónico que recibiste.

¿Cómo funciona?

  1. Solicita la recuperación de contraseña desde la aplicación móvil
  2. Revisa tu correo electrónico
  3. Haz clic en el enlace de recuperación del correo
  4. Serás redirigido automáticamente a la página de restablecimiento

Nota: No puedes acceder directamente a esta página. Debes usar el enlace único que se envió a tu correo.
